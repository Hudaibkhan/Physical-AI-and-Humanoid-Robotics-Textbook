# Quickstart Guide: Physical AI & Humanoid Robotics Textbook

**Feature Branch**: `001-physical-ai-book`
**Created**: 2025-12-05

## Overview

This guide provides essential steps to set up the development environment for contributing to the Physical AI & Humanoid Robotics Textbook project, and for students to get started with the practical exercises.

## 1. Prerequisites (Windows 10/11 - PowerShell-based Workflow)

- [ ] Ensure Windows 10/11 is installed.
- [ ] Install Python 3.x (via Chocolatey or official installer).
- [ ] Install Git (via Chocolatey or official installer).
- [ ] Install Node.js and npm (for Docusaurus).
- [ ] Install Visual Studio (with C++ development tools for ROS 2).

## 2. ROS 2 Humble Installation (Windows MSI)

- [ ] Follow official ROS 2 documentation for MSI installation of Humble Hawksbill.
- [ ] Verify ROS 2 environment sourcing in PowerShell.

## 3. Docusaurus Setup

- [ ] Clone the textbook repository.
- [ ] Navigate to the project root and run `npm install`.
- [ ] Start the Docusaurus development server with `npm start`.
